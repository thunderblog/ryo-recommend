# ファイル名
ryo-recommend.html

## 概要
両学長がYouTubeでオススメしている[第19回 【リピート確定！】学長が選ぶ大阪の美味しい飲食店25選【良いお金の使い方編】](https://www.youtube.com/watch?v=AgMTMvI7caQ)で紹介されている店舗をマッピングしました。  

## 使い方
学長オススメの飲食店25選の情報を下記情報を元にポップアップしています。  
  
* 店舗名
* 住所
* 最寄駅／料理のジャンル
* YouTubeの動画で紹介されているチャプターへ移動するURL

概要欄で紹介した動画の中で、お店を紹介しているチャプターへ直接ジャンプできます。  
気になったお店があれば、両さんの解説を聞いてみてください。

このHTMLファイルはPythonの「folium」というライブラリを使用して描画しています。
